# CorePrep | Math & Language Study Dashboard

## What is this platform?
CorePrep is a distraction-free, high-utility Study Planner and Curriculum Tracker tailored for rigorous Math and English language preparation. It serves as a unified curriculum dashboard that maps out learning paths directly to the best curated web resources, integrating study flows (Learn -> Practice -> Verify) and essential tools like the embedded Desmos graphing calculator.

## Who uses it?
The target users are students preparing for high-stakes, competitive academic and language milestones, typically over a structured 2-year timeline. Specifically, this platform is for students preparing for:
*   **SAT (Digital)**: Requiring high-speed, accurate algebra, and data interpretation.
*   **A-Levels**: Requiring rigorous, structured mathematical reasoning and a strong Calculus foundation.
*   **OSN (Indonesian Math Olympiad)**: Requiring high-level creative problem-solving, modular arithmetic, combinatorics, and rigorous proofs.
*   **IELTS**: Requiring systematic English language development from general vocabulary to academic writing and timed simulations.

## Study Case
CorePrep was built to address several core problems students face during self-study:
*   **The "Muddy Foundation" Trap**: Students often rush into advanced SAT, A-Level, or OSN math topics before mastering foundational arithmetic. CorePrep provides a clear, visible sequence of topics to trace prerequisites.
*   **The Arithmetic & Fraction Bottleneck**: CorePrep prioritizes and flags crucial bottleneck topics (e.g., Fractions, Negative Numbers, PEMDAS) to ensure mastery before moving forward.
*   **Overlapping but Disparate Goals**: Preparing for SAT, A-Levels, OSN, and IELTS simultaneously is overwhelming. CorePrep provides a unified, clean syllabus manager grouping topics logically under Math Tiers and Language Tracks.
*   **Time Management Breakdown**: Maintaining a daily routine of 1–1.5 hours of study and balancing Math (60%) and English (40%) is challenging. CorePrep includes a Daily Log Sidebar and Schedule Planner to track hours, problems solved, and recommended daily focus.

## How to run it
To get started with the CorePrep dashboard on your local machine:

1. **Clone the repository:**
   ```bash
   git clone https://github.com/RinggaDev/CORE-PREP-IO.git
   ```

2. **Navigate to the directory:**
   ```bash
   cd CORE-PREP-IO
   ```

3. **Run the application:**
   Since this is a vanilla HTML/CSS/JS application, no build steps or package installations are required. Simply open the `index.html` file in your preferred modern web browser.
   *Double-click `index.html` from your file explorer, or use a live server extension if you are using an editor like VS Code for hot-reloading.*   
   
   or

4. **How to Start the Server Manually in the Future**
If the server is ever stopped and you need to start it again, open a terminal in the project directory (`d:\3. Code Project\MATH`) and run one of the following commands:

* **Using Python** (recommended/installed):
  ```powershell
  python -m http.server 8000
  ```
* **Using Node/NPM**:
  ```powershell
  npx http-server -p 8000
  ```

Once started, open [http://localhost:8000/](http://localhost:8000/) in your browser.
